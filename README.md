# Sales Data Analysis Dashboard

## Objective

This project analyzes sales data using Power BI and presents key business insights through visual storytelling.

## Tools Used

* Power BI
* CSV Dataset
* GitHub

## Data Cleaning

* Verified data types
* Checked for duplicate records
* Created Total Sales column
* created Total revenue mesure
* created total quantity sold measure
* created total orders measure

## Visualizations
* Total Revenue KPI
* Total Orders KPI
* Total Quantity Sold KPI
* Sales by City
* Sales by Product
* Sales by Purchase Type
* Payment Method Analysis
* Manager Performance Analysis

## Key Insights
Lisbon generated the highest sales revenue.
Burgers were the best-selling product.
Online purchases contributed the largest share of sales.
Credit Card was the most frequently used payment method.
Revenue was concentrated among a few top-performing managers.


Interview answers
### 1. What is the importance of data visualization?

Data visualization helps transform raw data into easy-to-understand charts and graphs. It allows users to identify trends, patterns, outliers, and insights quickly, making decision-making more effective.


### 2. When do you use a pie chart vs bar chart?

* **Pie Chart:** Used to show proportions or percentages of a whole when there are only a few categories.
* **Bar Chart:** Used to compare values across different categories and is better when precise comparisons are needed.

### 3. How do you make visualizations more engaging?

* Use appropriate chart types.
* Keep the design clean and uncluttered.
* Use consistent colors and formatting.
* Highlight key insights and trends.
* Add titles, labels, and context to help users understand the data.


### 4. What is data storytelling?

Data storytelling is the process of combining data, visualizations, and narrative explanations to communicate insights and support decision-making. It helps audiences understand not just what happened, but why it matters.


### 5. How do you avoid misleading visualizations?

* Use accurate scales and axes.
* Avoid manipulating chart dimensions.
* Clearly label data and units.
* Choose the correct chart type for the data.
* Present data objectively without hiding important information.


### 6. What are best practices in dashboard design?

* Keep the layout simple and organized.
* Display the most important KPIs at the top.
* Use consistent colors and fonts.
* Avoid unnecessary visuals and clutter.
* Ensure charts are easy to read and interpret.
* Focus on actionable business insights.

For this project, I used **Microsoft Power BI** to clean data, create visualizations, build interactive dashboards, and present business insights. I am also familiar with visualization tools such as, **Microsoft Excel**, and Python libraries.
